## `src/app/layout.tsx` — root layout with Geist correctly wired

```tsx
import type { Metadata } from 'next'
import { GeistSans } from 'geist/font/sans'
import { GeistMono } from 'geist/font/mono'
import { ThemeProvider } from '@/components/theme-provider'
import './globals.css'

export const metadata: Metadata = {
  title: { default: 'App', template: '%s · App' },
  description: 'Your application',
}

export default function RootLayout({ children }: { children: React.ReactNode }) {
  return (
    <html
      lang="en"
      suppressHydrationWarning
      className={`${GeistSans.variable} ${GeistMono.variable}`}
    >
      <body className="min-h-screen bg-background font-sans antialiased">
        <ThemeProvider
          attribute="class"
          defaultTheme="system"
          enableSystem
          disableTransitionOnChange
        >
          {children}
        </ThemeProvider>
      </body>
    </html>
  )
}
```

**Critical:** `GeistSans.variable` sets `--font-geist-sans` on `<html>`. `font-sans` in `tailwind.config.ts` maps to that variable. Both are required — one without the other = system font fallback.

## `src/components/page-wrapper.tsx` — animate pages on load

```tsx
export function PageWrapper({ children }: { children: React.ReactNode }) {
  return (
    <div className="animate-fade-in">
      {children}
    </div>
  )
}
```

Use in every page:
```tsx
export default function SomePage() {
  return (
    <PageWrapper>
      <div className="space-y-8">...</div>
    </PageWrapper>
  )
}
```

## Animation usage guide

### Staggered cards (use `[animation-delay]` style)
```tsx
{items.map((item, i) => (
  <div
    key={item.id}
    className="animate-fade-in-up"
    style={{ animationDelay: `${i * 60}ms`, animationFillMode: 'both' }}
  >
    <Card>...</Card>
  </div>
))}
```

### Sidebar links — slide in on mount
```tsx
<Link className="... transition-all duration-150 hover:translate-x-0.5">
```

### Cards — lift on hover
```tsx
<Card className="transition-all duration-200 hover:shadow-md hover:-translate-y-0.5 cursor-pointer">
```

### Buttons — already handled by shadcn, but add scale:
```tsx
<Button className="... active:scale-95 transition-transform">
```

### Theme toggle — Sun/Moon already animate via Tailwind classes in the component.

### Skeleton shimmer (use instead of plain Skeleton when loading feels too static):
```tsx
<div className="h-8 w-32 rounded-md bg-muted animate-shimmer bg-gradient-to-r from-muted via-muted-foreground/10 to-muted bg-[length:200%_100%]" />
```

### Loading spinner:
```tsx
import { Loader2 } from 'lucide-react'
<Loader2 className="h-4 w-4 animate-spin" />
```
